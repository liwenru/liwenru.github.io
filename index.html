更换hexo主题

By lee

发表于 2018-10-17

  1. hexo主题地址：https://hexo.io/themes/，
    可以在上面选择自己喜欢的主题
  2. 将主题克隆到 themes 目录下，地址在如下图，，
    $ git clone https://github.com/wuchong/jacman.git themes/jacman
  3. 修改主题在_config.yml文件里找到theme,改为你想用的主题名称
